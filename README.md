# Linux Admin Toolkit

Operational Linux administration toolkit for Ubuntu/OpenClaw infrastructure.

Designed for:
- Ubuntu Server 24.04 LTS
- OpenClaw runtime management
- NucBox / mini-PC deployments
- Oracle Linux / Ubuntu VPS deployments
- Repeatable infrastructure bootstrap
- SSH-secured headless server administration

---

# Objectives

This repository serves as the operational control plane for Linux/OpenClaw systems.

Primary goals:
- Standardized Linux host setup
- Repeatable deployment process
- OpenClaw operational management
- SSH/network validation
- Runtime auditing
- Nginx/SSL administration
- Cross-server consistency
- GitHub-based infrastructure replication

---

# Current Environment

Validated on:

- Ubuntu 24.04.4 LTS
- GMKtec NucBoxG3S
- Node.js v22.x
- OpenSSH Server
- OpenClaw runtime
